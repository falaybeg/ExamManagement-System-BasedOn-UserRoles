In this project was written a Exam Management System which was my first internship project. During the project I learned C#, Ado.Net, Sql and creating WinForm application. 
The project has 7 user role which are as following:

- Admin 
- Province manager
- Vice Province Manager 
- District Manager
- Building Manager 
- Vice Building Manager 
- Supervisor
